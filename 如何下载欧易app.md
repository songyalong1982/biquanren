想下载欧易app但不知道如何操作？我用欧易已经3年多了，安卓和苹果的下载方法都试过，直接告诉你最简单的步骤。

# **欧易官网在哪找？**

首先在浏览器里输入 **okx.com** ，这是欧易的官方网站。进入网站后，你会看到右上角有三条横杠的菜单按钮，点击它。

在下拉菜单里找到「下载欧易APP」这个选项，点击进去就能看到下载页面了。

# **安卓手机怎么下载欧易app？**

如果你用的是安卓手机，下载就很简单了。在下载页面直接点击「下载欧易app」按钮，系统会自动下载APK安装包到你手机上。

下载完成后，找到APK文件直接安装就行。可能会提示「未知来源应用」，允许安装就可以了。我第一次下载时也遇到这个提示，完全不用担心。

# **苹果手机用户怎么下载？**

苹果用户稍微麻烦一点，因为中国区App Store没有欧易app，需要用海外的Apple ID才能下载。

**方法一：自己注册海外Apple ID**

1. 在欧易官网的下载页面，你能看到Apple ID注册流程的链接
2. 跟着流程一步步注册一个台湾或香港区的Apple ID
3. 注册时记得选择台湾或香港作为国家/地区

**方法二：直接购买现成的Apple ID**

如果嫌注册麻烦，可以在Google搜索「台湾apple ID购买」，会有很多商家在卖。我看到一个台区Apple ID账号只要12块钱，还算便宜。

1. 选择一家没有广告、看起来靠谱的商家
2. 输入你的邮箱地址
3. 选择购买数量（一般买一个就够了）
4. 完成支付，商家会把账号密码发给你

# **怎么用海外Apple ID下载欧易？**

拿到海外Apple ID后，就可以下载欧易app了。**注意：一定要在App Store里切换账号，千万别在设置里换登**，这点很重要！

1. 打开App Store应用
2. 点击右上角的小人头像
3. 滑到页面最底部，点击「登出」
4. 用刚买的或刚注册的海外Apple ID登录
5. 在搜索框输入「OKX」进行搜索
6. 搜索结果会出现几个选项，选择第二个进行下载

为什么不选第一个？因为第一个通常是广告，第二个才是真正的欧易官方app。我之前就点错过，下载了个假的，后来才发现。
